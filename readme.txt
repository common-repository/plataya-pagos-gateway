== Description ==
Adds <b>Plata Ya Credito</a> payment method to available payment methods

<b>Plata Ya Credito</b> is a new means of payment for you to finance all your online sales in up to 12 fixed installments, without the need to use credit or debit cards.

== Installation ==
= Requisites =
* WooCommerce Plugin
* LAMP (Linux, Apache, MySQL, PHP)
* SSL Certificate

Install automatically from the "Plugins" section of WordPress.

1. Access "Plugins" from the navigation side menu of your WordPress administrator.
2. Once inside Plugins, click on "Add New" and search for 'Plata Ya Credito payments for WooCommerce' in the WordPress Plugin list
3. Click on "Install."

Ready! It will be in the "Installed Plugins" section and from there you can activate it.
